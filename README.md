    <h1>Curso de Python: PIP y Entornos Virtuales</h1>

    <p>Este proyecto es un curso introductorio sobre el manejo de PIP (Python Package Installer) y entornos virtuales en Python. Aprenderás cómo usar PIP para instalar y gestionar paquetes de Python, así como la importancia y los beneficios de trabajar con entornos virtuales para mantener proyectos Python limpios y organizados.</p>

    <h2>Instalación</h2>

    <ol>
        <li>Clona este repositorio en tu máquina local:
            <pre><code>git clone https://github.com/tu_usuario/nombre_proyecto.git</code></pre>
        </li>
        <li>Accede al directorio del proyecto:
            <pre><code>cd nombre_proyecto</code></pre>
        </li>
        <li>Crea y activa un entorno virtual (se recomienda, pero opcional):
            <pre><code>python3 -m venv venv
source venv/bin/activate</code></pre>
        </li>
        <li>Instala las dependencias del proyecto:
            <pre><code>pip install -r requirements.txt</code></pre>
        </li>
    </ol>
